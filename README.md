# TechShellProject
C program called techshell.c that acts like a bash shell. That is, it should repeatedly allow a user to enter input that gets interpreted as a shell command.

Created by Ethan Flanagan, Adam Young, Jayline Ealy

Responsibilities: Each one of the members made their own version of the project, and the best one was chosen for main.

Works only on Linux Software


Standard terminal, uses bash commands such as (cd, ls, chmod, ect.)

How to run program

'''bash
git clone https://github.com/AdamYoung19/TechShellProject.git
cd TechShellProject

gcc ShellProject.c -o TechShell

'''
